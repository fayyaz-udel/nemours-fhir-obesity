Early Weight Gain Prediction Tool

What to say to child's family

I’d like to work together to slow child’s weight gain. Are you open to talking about some healthy lifestyle changes for your family?

If yes Great! To get started, I’m going to ask you some questions about child’s behaviors, sleep, activity, and eating so we can see where there are opportunities to set some goals. I have some great resources that I can give you at the end of this visit to get started with whatever goals we set together today!

If no I understand. We can talk about this at a future visit when you’re ready. Could I give you a few resources with tips for all kids to be healthy”?
